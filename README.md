🚗Descriere
-Mechanic Shop este o aplicatie software dezvoltata pentru gestionarea unui service auto. Proiectul permite administrarea clientilor, masinilor si operatiilor specifice unui shop auto printr-o interfata bazata pe consola. Scopul proiectului a fost aplicarea conceptelor de programare si structurare a datelor intr-un scenariu real, combinand logica software cu un caz practic din domeniul auto.

⚙️ Functionalitati
-Adaugare clienti si masini; Stergere si modificare obiectelor adaugate; Gestionare reparatii; Salvare si incarcare date din baza de date; Sistem de meniu interactiv

🧠 Tehnologii utilizate
-Language: C++; Programare orientata pe obiecte; File handling (citire/scriere in baza de date)

🎯 Obiectivele proiectului
-Aplicarea conceptelor de programare intr-un proiect real; Simularea unui sistem de management pentru un shop auto; Organizarea eficienta a codului si a datelor; Dezvoltarea abilitatilor de problem solving
